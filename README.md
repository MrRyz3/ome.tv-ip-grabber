# Installation Guide

## 1. Install Tampermonkey

First, install **Tampermonkey** from the official website:

https://www.tampermonkey.net/

**Only download Tampermonkey from the official website.**

**Warning:** Do not run random scripts you find on GitHub or other websites. Always review the source code before installing any userscript.


## 2. Add the Tampermonkey Extension

Install the Tampermonkey extension for your browser.

### Chromium-Based Browsers (Chrome, Edge, Brave, Opera, etc.)

1. Click the **Extensions** icon in your browser toolbar.
2. Select **Manage Extensions**.
3. Enable **Developer Mode** in the top-right corner.
4. Open the **Tampermonkey** extension details page.
5. Scroll down and enable **Allow User Scripts**.

Tampermonkey is now ready to use.


## 3. Install the Script

1. Click the **Extensions** icon.
2. Open **Tampermonkey**.
3. Click **Create a New Script**.
4. Delete all existing content in the editor.
5. Copy the script from this repository and paste it into the editor.
6. Save the script using **Ctrl + S** or by clicking **File → Save**.


## 4. Done!

Open **Ome.tv** and enjoy.
